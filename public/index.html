Gif countdown generator Using node

The very simple app I have created allows you to generate a countdown timer animated gif depending on the URL parameters you provide. View the code.

URL Parameters

  • time*: Date & time when your countdown will end [e.g. 2016-06-24T20:35]
  • width: width in pixels [defaults to 400]
  • height: width in pixels [defaults to 200]
  • frames: number of frames (also number of seconds) the countdown will run before looping [defaults to 30]
  • textColor: hex colour code for the text [defaults to FFFFFF]
  • bgColor: hex colour code for the background [defaults to 000000]
  • digitFontFamily: Font family for the digit part [defaults to NotoSans]
  • digitFontSize: Font size or digit part.
  • unitLocale: The unit string for units. [default to "Days_Hours_Mins_Secs"]
  • unitFontFamily: Font family for the digit part [defaults to NotoSans]
  • unitFontSize: Font size or unit part.
  • passedMsgFontFamily: Font family for the digit part [defaults to NotoSans]
  • passedMsgFontSize: Font size for the passed message.
  • passedMsg: The message to show when the time is passed.

* required.

Generate Examples Triggers a download

  • Basic: /generate?time=2025-09-24T20:35
  • Custom dimensions: /generate?time=2025-09-24T20:35&width=300&height=150
  • Custom colours: /generate?time=2025-09-24T20:35&bgColor=264D00&textColor=66CC00
  • Custom name & frames: /generate?time=2025-09-24T20:35&frames=20
  • Custom passed message: /generate?time=2025-09-24T20:35&frames=20
